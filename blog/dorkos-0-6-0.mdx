DorkOS 0.6.0 introduces a guided first-time setup experience that discovers your existing AI agents, creates recurring schedules, and gets you productive in minutes. This release also overhauls remote access and adds research library curation.

## Highlights

**Guided Onboarding** — New users are walked through a three-step setup: agent discovery scans your machine for AI-configured projects, Pulse presets create recurring schedules targeting your registered agents, and a completion screen summarizes everything that was configured.

**Remote Access Redesign** — Multi-tab sync, a refreshed UX, and CLI QR code support make accessing DorkOS remotely smoother than ever. Curl install script, tabbed UI, and Homebrew tap round out the installation story.

**Research Library Curation** — A new curation workflow processes, prunes, and consolidates research artifacts with structured frontmatter and lifecycle management.

## All Changes

- Walk through first-time setup with guided agent discovery, Pulse presets, and animated onboarding flow
- Overhaul remote access with multi-tab sync, UX redesign, and CLI QR code

- Add research library curation with file reduction phases

DorkOS 0.40.0 refines the agent experience end-to-end — from a new verbosity-based personality system and lifecycle management actions to polished sidebar animations and scoped marketplace installs.

## Highlights

**Verbosity-Based Personality System** — Agent traits shift from a tone-based model to verbosity levels, giving operators more intuitive control over how agents communicate. The PersonalityRadar adapts to both light and dark themes with the new nebula utilities.

**Agent Lifecycle Management** — New lifecycle actions let you manage agents directly from the Agent Hub with split navigation, context menus on agent chips, and success toasts for deny/unblock workflows.

**Scoped Marketplace Installs** — The marketplace Toolkit tab now leads with skills and supports scoped installs, so packages land exactly where you need them in your project.

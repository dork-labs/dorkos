# Installation

DorkOS runs in several modes. Pick the install method that fits your workflow.

## Prerequisites

<Callout type="info">
  All installation methods require [Node.js](https://nodejs.org/) 18 or later and a Claude API key from the [Anthropic Console](https://console.anthropic.com/).
</Callout>

## Choose Your Install Method

<Tabs items={['One-liner (Recommended)', 'npm', 'Homebrew', 'Obsidian Plugin', 'Self-Hosted']}>

<Tab value="One-liner (Recommended)">

The fastest way to get started. A single command that checks your environment, installs DorkOS, and optionally runs the setup wizard.

<Steps>
<Step>

### Install DorkOS

```bash
curl -fsSL https://dorkos.dev/install | bash
```

The install script checks for Node.js 18+, installs DorkOS via npm, and optionally runs the setup wizard.

</Step>
<Step>

### Set your API key

```bash
export ANTHROPIC_API_KEY="your-key-here"
```

To persist this, add it to your shell profile (`~/.bashrc`, `~/.zshrc`, etc.).

</Step>
<Step>

### Start DorkOS

```bash
dorkos
```

The server starts on port 4242 and opens your browser automatically.

</Step>
</Steps>

<Callout type="info">
  **CI/Automation:** Use `--no-prompt` to skip interactive prompts:
  ```bash
  curl -fsSL https://dorkos.dev/install | bash -s -- --no-prompt
  ```
  Pin a specific version: `curl -fsSL https://dorkos.dev/install | bash -s 1.2.3`
</Callout>

</Tab>

<Tab value="npm">

Install the `dorkos` package globally with your preferred package manager.

<Steps>
<Step>

### Install the package

```bash tab="npm"
npm install -g dorkos
```

```bash tab="pnpm"
pnpm add -g dorkos
```

```bash tab="yarn"
yarn global add dorkos
```

```bash tab="bun"
bun add -g dorkos
```

</Step>
<Step>

### Set your API key

```bash
export ANTHROPIC_API_KEY="your-key-here"
```

To persist this, add it to your shell profile (`~/.bashrc`, `~/.zshrc`, etc.).

</Step>
<Step>

### Verify the installation

```bash
dorkos --version
```

</Step>
<Step>

### Start DorkOS

```bash
dorkos
```

The server starts on port 4242 and opens your browser automatically.

</Step>
</Steps>

<Callout type="info">
  You can pass flags to customize startup: `dorkos --port 8080 --dir ~/projects/my-app`. Run `dorkos --help` for all options.
</Callout>

</Tab>

<Tab value="Homebrew">

Install via Homebrew on macOS or Linux. Homebrew handles Node.js as a dependency automatically.

<Steps>
<Step>

### Install DorkOS

```bash
brew install dorkos-ai/tap/dorkos
```

</Step>
<Step>

### Set your API key

```bash
export ANTHROPIC_API_KEY="your-key-here"
```

To persist this, add it to your shell profile (`~/.bashrc`, `~/.zshrc`, etc.).

</Step>
<Step>

### Start DorkOS

```bash
dorkos
```

</Step>
</Steps>

<Callout type="info">
  Update via `brew upgrade dorkos`.
</Callout>

</Tab>

<Tab value="Obsidian Plugin">

Run DorkOS directly inside [Obsidian](https://obsidian.md/) as a sidebar panel, with full access to your vault files.

<Steps>
<Step>

### Open Obsidian settings

Go to **Settings** then **Community Plugins** and make sure community plugins are enabled.

</Step>
<Step>

### Search for DorkOS

Click **Browse** and search for **DorkOS** in the community plugin directory.

</Step>
<Step>

### Install and enable

Click **Install**, then toggle the plugin on. DorkOS appears as a sidebar view.

</Step>
<Step>

### Configure your API key

Open the DorkOS plugin settings and enter your Anthropic API key.

</Step>
</Steps>

<Callout type="warn">
  The Obsidian plugin requires Node.js 18+ installed on your system. It uses the same Claude Agent SDK as the CLI but runs in-process via the DirectTransport adapter.
</Callout>

</Tab>

<Tab value="Self-Hosted">

Clone the repository and run the full development stack. Useful for contributing or customizing DorkOS.

<Steps>
<Step>

### Clone the repository

```bash
git clone https://github.com/dork-labs/dorkos.git
cd dorkos
```

</Step>
<Step>

### Install dependencies

```bash tab="npm"
npm install
```

```bash tab="pnpm"
pnpm install
```

```bash tab="yarn"
yarn install
```

```bash tab="bun"
bun install
```

</Step>
<Step>

### Configure environment

Create a `.env` file in the repository root:

```bash
ANTHROPIC_API_KEY=your-key-here
```

</Step>
<Step>

### Start the development server

```bash
npm run dev
```

This starts both the Express API server and Vite dev server. The interface is available at [http://localhost:3000](http://localhost:3000) with hot module replacement.

</Step>
</Steps>

<Callout type="info">
  For production builds, run `npm run build` then `npm start` to serve the compiled app on port 4242.
</Callout>

</Tab>

</Tabs>

## Updating

<Tabs items={['One-liner / npm', 'Homebrew', 'Obsidian Plugin', 'Self-Hosted']}>

<Tab value="One-liner / npm">

Re-run the one-liner or update with your package manager:

```bash tab="One-liner"
curl -fsSL https://dorkos.dev/install | bash
```

```bash tab="npm"
npm install -g dorkos@latest
```

```bash tab="pnpm"
pnpm add -g dorkos@latest
```

```bash tab="yarn"
yarn global add dorkos@latest
```

```bash tab="bun"
bun add -g dorkos@latest
```

DorkOS checks for new versions on startup and shows a notification when an update is available. You can check your current version at any time:

```bash
dorkos --version
```

</Tab>

<Tab value="Homebrew">

```bash
brew upgrade dorkos
```

</Tab>

<Tab value="Obsidian Plugin">

Obsidian handles plugin updates automatically. To check manually, go to **Settings** then **Community Plugins**, find DorkOS in your installed plugins, and click **Update** if available.

</Tab>

<Tab value="Self-Hosted">

Pull the latest changes and rebuild:

```bash
git pull origin main
npm install
npm run build
```

</Tab>

</Tabs>

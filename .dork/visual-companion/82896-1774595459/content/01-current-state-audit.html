Issues Identified

Structural Issues

  • Enable toggle buried at the bottom — should be the hero control
  • Settings vanish when connected (can't adjust passcode/domain)
  • Onboarding illustration wastes vertical space for expert users
  • 570 lines / 14+ state variables — component is overloaded
  • Mobile drawer not full-width (max-w-md leaks to drawer)

Missed Opportunities

  • No progressive disclosure — all config visible at once
  • "Starting..." has no step-by-step progress feedback
  • No connection quality context (latency hidden in tooltip)
  • Copy feedback uses icon swap (good!) but passcode uses toasts
  • No scope transparency — doesn't explain what remote access enables
  • Token input has no format validation or paste detection

### Added

- MCP servers can now ship interactive apps that render right in the chat. When a tool returns a `ui://` app (MCP Apps, SEP-1865), DorkOS shows it inline as a live, sandboxed panel — and you can pop it out to the full canvas. Works with Claude Code sessions today.
- The first app from any given MCP server asks before it renders, and remembers your choice per server, so a trusted server never asks twice.
- Apps run locked down: they cannot reach your session, files, cookies, or credentials, external links route through the same "open external link?" confirmation the rest of the app uses, and an app cannot run tools on your behalf.

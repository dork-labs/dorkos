### Added

- The external MCP server (`/mcp`) now exposes read-only resources alongside its tools: `dorkos://sessions`, `dorkos://agents`, and `dorkos://skills` list catalogs, plus `dorkos://sessions/{id}`, `dorkos://agents/{id}`, and `dorkos://skills/{name}` item templates, so MCP hosts that render resource catalogs directly can browse DorkOS data without a tool call

### Added

- Declarative widget tier — dorkos-ui fences, catalog renderer, canvas widgets
- Agents can render native, interactive UI inline in chat. Emit a `dorkos-ui` fenced JSON block and it renders as a real widget — stat cards, tables, lists, charts, key/value pairs, badges, progress bars, images, forms, and more — woven into the reply instead of a code block. Malformed widgets degrade to a small error card, so chat never breaks.
- Widgets can also open in the canvas: `control_ui open_canvas` accepts a `widget` content type, rendering the same widget with room to breathe.
- The widget catalog is taught to every runtime (Claude Code, Codex, OpenCode), so any agent can use it.

### Added

- **Extensions can now add their own tabs to the sidebar.** An extension that contributes a `sidebar.tabs` view now shows up as a real tab in the agent sidebar, alongside the built-in Overview, Sessions, Schedules, and Connections — so the Linear Ops shape's Linear tab actually appears once you enable it. A contributed tab opens its own panel, keeps its place in your workspace across reloads, and quietly steps aside back to Overview if you later remove the extension. Cmd/Ctrl+1–4 still select the four built-in tabs.

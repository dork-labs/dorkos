### Fixed

- Shared blog links now show a proper preview card. Every blog post gets its own preview image, so links pasted into Slack, Discord, LinkedIn, or X show a real card instead of a bare link (DOR-344)
- Link previews on X now match the rest of the page. Twitter cards used to fall back to one generic sitewide title and description; every page now sets its own (DOR-344)
- The sitemap now reports honest "last updated" dates: blog posts use their post date and docs pages use the date the page was last changed, instead of claiming every page changed today (DOR-344)

### Changed

- Social preview images now use the real DorkOS brand font and share one set of building blocks, so previews look consistent across the blog, marketplace, and feature pages (DOR-344)
- Blog post previews now show a reading-time estimate and when the post was last updated, not just when it was first published. The story page also got its own preview card and link details for the first time (DOR-344)
- The security, privacy, terms, cookies, and telemetry pages now carry the same link-preview and canonical-URL details as the rest of the site (DOR-344)

### Added

- Agents can render four new inline widgets: a **timeline** for sequenced events (done / in-progress / upcoming), a **checklist** whose ticked items report back to the agent when you confirm, a **compare** table that highlights a recommended option, and a **star rating** with a review count.
- List widgets can now lead each row with a thumbnail image and show a right-aligned detail (like a price), and stat widgets can display a small inline sparkline of recent history.
